## arguments
- 所有函数中(除了箭头函数)都可用的局部变量
- [arg1,arg2,arg3]


##  普通函数箭头函数  区别
- 箭头函数不支持重名形参，普通函数可以
- 箭头函数没有 this,  它的 this 指向定义时 所处的上下文(外部函数) this 
- 箭头函数不能通过 .call .apply ...改变箭头函数的 this
- 箭头函数没有 .prototype 
- 箭头函数不能作为构造函数
- 箭头函数没有 arguments, 
- 箭头函数 没有 new.target
