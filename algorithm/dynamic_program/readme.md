老虎吃羊  

500只老虎，1只羊
都吃草 
老虎可以吃羊 ， 一只老虎吃一只羊
吃完羊后，这只老虎会变成羊
会不会吃掉这只羊？
找规律后，可以递归，或类推 
  n  



1. 吃什么羊，虎变羊 被吃
2. 吃， 羊肉好吃  
3. 不吃，羊那么可爱，吃干嘛， 还可以多个伙伴 
  一只老虎 (1)  一只羊  吃
  两只老虎 (2)  一只羊  不吃 
  三只老虎 (3)  一只羊  吃
  四只老虎 (4)  一只羊  不吃
  五只老虎 (5)  一只羊  吃
  奇数吃， 偶数就不吃    500  不吃


动态规划:   大事化小，小事化了，  找到规律，  
最优子结构  上一次不吃，下一次就吃，
边界 n = 1 吃    n=2 不吃  递归子结构
状态转移  F(n) = !F(n-1)


楼梯只有12阶，一步走1阶或走2阶，请问走完楼梯多少种走法？

1. 边界 n = 1  1种，走1步   n = 2  2种
2. 子结构  n = 12 时，有多少种走法
n -1 = 11  1步  +  n - 2 = 10  2步 
3. 状态转移公式
   F(n) = F(n-1) + F(n-2)
